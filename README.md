# WormLocker 2.0 Ransomware Analysis

## Project Overview

This repository contains the deliverables of a comprehensive malware analysis project focused on **WormLocker 2.0**, a ransomware family targeting Windows systems.

The project combines malware reverse engineering, dynamic analysis, digital forensics, incident response, and threat intelligence techniques to understand the ransomware's behavior, infection mechanisms, persistence methods, and impact on compromised systems.

The analysis was conducted in a controlled and isolated laboratory environment to safely observe the malware's execution and collect forensic evidence.

## Project Objectives

* Analyze the behavior of a real-world ransomware sample.
* Identify indicators of compromise (IOCs).
* Investigate persistence and privilege escalation mechanisms.
* Study file encryption and system modification techniques.
* Perform memory and forensic analysis of an infected system.
* Develop incident response and containment procedures.
* Document findings for defensive cybersecurity operations.

## Key Findings

* Multi-stage ransomware execution chain.
* Windows persistence through registry modifications.
* Screen-locking and file encryption capabilities.
* Deployment of malicious supporting components.
* System-level modifications affecting user access.
* Discovery of operational security weaknesses within the malware.
* Identification of artifacts useful for detection and investigation.
